Better PaperClip
github
Better Paperclip is an updated UI/UX of Brooklyn College's printing service Papercut. This website features a simplified UI that features larger buttons and easy navigation to common services such as replenishing balance and uploading a file to print. Worked in a team of 5 and surveyed current alumni on their experiences with current printing services available to them.
Gear Gates
github
Gear Gates is a 2d pixel game whose objective is to fix a town's electrical problems by collecting gears to fix the problem. Java processing was used to develop the movement and boundary logic. implemented event handlers for movement, picking up gears, and level logic.
